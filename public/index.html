Kurulum Sihirbazı

Sistemi 3 adımda hazırla. Her şey yerel — bilgiler sadece bu makinede saklanır.

1
API Anahtarları
Tasarım üretimi ve SEO için gerekli. Dolduktan sonra sunucu otomatik kullanır.
bekliyor
wiro.ai → kayit ol → API Keys → Create Key. Goruntu uretimi icin kullanilir.
openrouter.ai → Keys → Create Key. SEO baslik ve etiket uretimi icin kullanilir. Ucretsiz krediler mevcuttur.
2
Tarayıcı (CDP)
Etsy ve Pinterest için gerçek Chrome gerekir. Otomatik algılanır.
bekliyor
Chrome, Edge, Brave veya Opera GX desteklenir. "Otomatik Bul" genellikle dogru yolu bulur.
Varsayilan 9333. Baska bir program bu portu kullaniyorsa degistir.
Onemli: CDP browser ayri profil kullanir (data/browser-profile/). Normal Chrome'undaki Etsy/Pinterest login burada YOK. "Tarayiciyi Ac" → acilan yeni pencerede etsy.com ve pinterest.com'a tekrar login ol. Bu pencere arka planda acik kalmali.
3
Etsy Şablonu
Şablon listing ID yeni ürünleri hızlı kopyalamak için.
bekliyor
Yeni urunler bu sablondan kopyalanir. Nasil bulunur: CDP browser'da etsy.com'a login ol → Etsy → Your Shop → Listings → kopyalanacak urunu ac → URL'deki rakam: /listing/1234567890/...
Hatirla: Etsy login'i CDP browser'da olmali. Sidebar > Tarayici Bagla > acilan pencerede login olduktan sonra sablon ID'ni gir.
4
Alura SEO Eklentisi
Etsy listinglerden tag/kelime cekmek icin gerekli Chrome eklentisi.
bekliyor
Etsy listinglerden SEO etiketi cekmek icin kullanilir. Kurulum:

1. Once sol alttaki Tarayici Bagla butonuyla CDP browser'i ac.
2. Tarayicida Ac butonuyla Alura'nin Chrome Web Store sayfasini ac.
3. CDP browser'da "Add to Chrome" tikla, eklentiyi yukle.
4. Otomatik Kontrol Et bas → sistem Etsy'de listing acip Alura'yi tespit eder (~15sn).

Tarayici (CDP)
Chrome DevTools Protocol — Etsy/Pinterest otomasyonu icin gerekli
Varsayilan: 9333. Cakisma varsa degistir.
Bos birakilirsa otomatik algilama kullanilir.
Etsy Hesabi
CDP tarayici uzerinden otomatik giris icin cookie
Bagli Degil
Nasil alinir: etsy.com'a giris yap → F12 → Console → asagidaki kodu yapistir → sonucu buraya yapistir:
copy(JSON.stringify(document.cookie.split(';').map(c=>{let[n,...v]=c.trim().split('=');return{name:n,value:v.join('='),domain:'.etsy.com'}})))
Pinterest Hesabi
Otomatik pin icin cookie verisi
Bagli Degil
Ayni kod, sadece domain'i .pinterest.com olarak degistir.
copy(JSON.stringify(document.cookie.split(';').map(c=>{let[n,...v]=c.trim().split('=');return{name:n,value:v.join('='),domain:'.pinterest.com'}})))
